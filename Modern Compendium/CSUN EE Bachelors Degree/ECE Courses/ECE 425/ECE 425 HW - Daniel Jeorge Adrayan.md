1. (a)( 5pts ) Write a void function called SSI2_Init that configures the SSI2 module with the following parameters.
	• SPI Format: Freescale SPI
	• Data Length: 8 bits
	• SPI Mode 3 (CPOL = 1 and CPHA = 1)
	• Clock Source: System Clock
	• SCLK Frequency: 2.5 MHz
	• SPI Pins: PB4 (SCLK), PB5 (CS), PB6 (MISO), and PB7 (MOSI)
	(b) (5 pts) Suppose that you have a main program that is transmitting data using the
	SSI2 module. Draw the waveform for the following signals if the main program is
	transmitting one byte which has a value of 0xAA.
	• Chip Select (CS)
	• Serial Clock (SCLK)
	• Master Out, Slave In (MOSI)
	
```cpp
void SSI2_Init(void)
SYSCTL->RCGCSSI |= 0x04; //SSI2 Clock
SYSCTL->RCGCGPIO |= 0x02; // Port B clock 

GPIOB->AFSEL |= 0xF0; // Enable alternate function on PB4-7  
GPIOB->DEN |= 0xF0; // D enable on PB4-7  
  
GPIOB->PCTL &= ~0xFFFF0000;  
GPIOB->PCTL |= 0x22220000; // SSI2 pins  
  
SSI2->CR1 &= ~0x02; // Need to disable before config

 
//Config process   -----------
SSI2->CC &= ~0x0F;   // Clear Clock source to do System Clock Source 
SSI2->CPSR = 8; // Prescaler = 8  
  
SSI2->CR0 &= ~0xFF00;   // Clear SCR
SSI2->CR0 |= 0x0300;    // SCR = 3

SSI2->CR0 |= 0x0080;    // SPH = 1
SSI2->CR0 |= 0x0040;    // SPO = 1
SSI2->CR0 |= 0x07; // DSS = 8-bit, FRF = Freescale SPI  
  
SSI2->CR1 &= ~0x01; // Master mode  
SSI2->CR1 &= ~0x04; // Normal operation  
  
SSI2->CR1 |= 0x02; // Enable SSI2
```

The function should wait for the SSI Transmit FIFO to be ready before sending a byte of data. Assume that the module being used is SSI0.
• BSY (Bit 4)
• TNF (Bit 1)
(b) (5 pts) Suppose that you want to receive a byte of data from an SPI-based sensor using the SSI0 module. Write a function called SSI_Receive that takes no arguments and
returns an unsigned 8-bit value. This function should wait for the data to be available in the Receive FIFO and return the received byte.


a) BSY can be thought of as "Busy" acts as a busy flag that is high while SSI is busy transmitting data
0 - SSI idle
1 - SSI busy

TNF stands for Transmit Not Full, essentially it acts as a flag that stands high when the FIFO has space and can be written to. Other wise it would stand low to notify that it is unable to be written to.

```cpp
void SSI_Write(uint8_t data)  
{  
while((SSI0->SR & 0x02) == 0); // TNF  
SSI0->DR = data;  
  
while(SSI0->SR & 0x10); // Wait until BSY = 0  
}
```

```cpp
uint8_t SSI_Receive(void)
{
    while((SSI0->SR & 0x04) == 0);  // Wait FOR RECIEVE 
    return (uint8_t)(SSI0->DR);     // Read data and send
}
```
